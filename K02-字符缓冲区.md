# 字符缓冲区

在使用 `scanf` 函数时，可能会遇到如下问题：

```c++
printf("母亲身高（cm）：");
scanf("%lf", &moHeight);

printf("是否喜欢体育锻炼：（Y/N）");
scanf("%c", &sports);

printf("是否有良好饮食习惯：（Y/N）");
scanf("%c", &diet);
```

键盘输入完母亲身高后，会出现

```text
母亲身高（cm）：150
是否喜欢体育锻炼：（Y/N）是否有良好饮食习惯：（Y/N）
```

无法输入体育锻炼那一项的值，这是因为在上面输入完 150 后敲了回车，产生的换行符没有被变量`moHeight`拿走，还在字符缓冲区中，被下一个变量`sports`拿走了，导致 `sports` 就不用读取输入了

所以，需要清除掉这个产生的换行符，可以使用 `getchar()` 函数来吸收字符缓冲区的东西

所以，使用 `scanf()` 后最好加个 `getchar()`，即最后修改成

```c++
printf("母亲身高（cm）：");
scanf("%lf", &moHeight);
getchar();

printf("是否喜欢体育锻炼：（Y/N）");
scanf("%c", &sports);
getchar();

printf("是否有良好饮食习惯：（Y/N）");
scanf("%c", &diet);
getchar();
```
